# Contributing

Gracias por mejorar esta colección de prompts.

## Cómo proponer un nuevo prompt

1. Crea un archivo Markdown dentro de la carpeta correspondiente en `prompts/`.
2. Usa la plantilla de `plantillas/plantilla_prompt.md` o `plantillas/plantilla_prompt_gpt_5_5.md`.
3. Incluye un caso de uso concreto.
4. Agrega restricciones técnicas y de seguridad.
5. Define el formato de salida esperado.
6. Incluye reglas de detención.

## Convenciones de nombre

Usa nombres en minúsculas, sin espacios y con guiones bajos:

```text
01_nombre_del_prompt.md
```

## Reglas de calidad

Antes de abrir un pull request, verifica:

- El prompt tiene objetivo claro.
- La tecnología y versión están definidas.
- No incluye credenciales reales.
- Incluye criterios de éxito verificables.
- Incluye restricciones de seguridad si aplica.
- Indica cuándo pedir más información o declarar supuestos.

## Estilo

Mantén el lenguaje claro, directo y técnico.

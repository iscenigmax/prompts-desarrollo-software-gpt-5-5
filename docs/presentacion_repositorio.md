# Presentación del repositorio

## Problema

Muchos prompts para programación son demasiado vagos:

```text
Hazme una API.
Corrige este error.
Optimiza esta consulta.
```

Ese tipo de solicitud suele producir respuestas incompletas, con supuestos ocultos o sin validación.

## Solución

Este repositorio propone prompts estructurados para tareas de desarrollo de software usando ChatGPT/GPT-5.5.

Cada prompt define:

- Rol técnico.
- Estilo de respuesta.
- Objetivo concreto.
- Criterios de éxito.
- Restricciones.
- Formato de salida.
- Reglas para detenerse o pedir contexto.

## Para quién es

- Desarrolladores.
- Estudiantes.
- Equipos que trabajan con sistemas legacy.
- Analistas técnicos.
- Personas que usan ChatGPT para acelerar tareas de software sin perder control técnico.

## Mensaje corto para publicar

Colección de prompts estructurados para desarrollo de software con GPT-5.5. Incluye ejemplos para Node.js, SQL Server, C#, VB.NET, Web Forms, debugging, refactorización, testing, arquitectura y documentación técnica.
